Project Sanity Check


Project Sanity Check is an education index created for all provinces of Turkey. Using machine learning, the project measured the expected education indices of provinces for the year 2020, and compared with the actual index from the year 2020. This project thus aimed to show the effect of the Covid-19 pandemic on education in different Turkish provinces. The index and results are presented on an interactive, online map.
