Project Charisius


This is Project Charisius, my project at DCL lab at EPFL.

Project Charisius is a comparative benchmark research project dedicated to the study and analysis of various models and their resilience.

The project constitutes of 2 parts: Creating Highly-Optimized Gradient Aggregation Rules (Done!) and creating a reliable, benchmark metric (in progress).

You can see the optimized functions, the performance improvement in percentage, the metric creation and our model leaderboard in the website below!
